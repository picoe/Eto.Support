Remarks
This binding provides a way to get/set values for a particular object. This uses a IndirectBindingT as its logic to actually retrieve/set the values. This acts as a bridge between the IndirectBindingT and DirectBindingT so that you can utilize the DataValueChanged method. Typically, one would use the PropertyBindingT, or the [C:ObjectBinding{T,TValue}(T, string)] constructor to hook up this binding to a particular property of the specified object
