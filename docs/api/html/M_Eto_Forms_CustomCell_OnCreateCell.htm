Remarks
This is called multiple times usually for the number of visible and/or cached cells that are displayed. Note that you need to handle OnConfigureCell(CellEventArgs, Control) to set up the cell for a particular row of data as the control may be reused when scrolling the view. If you intend on creating a different type of cell control based on the item, you should also override GetIdentifier
