Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type ISubmenu. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Remarks
