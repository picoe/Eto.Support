Gets or sets the size of the control. Use -1 to specify auto sizing for either the width and/or height.

Remarks
Setting the size of controls is entirely optional as most controls will size themselves appropriately. When specifying a size, it will be used as the desired size of the control. The container will reposition and resize the control depending on the available size. For a Window, it is preferred to set the ClientSize instead, as various platforms have different sizes of window decorations, toolbars, etc.
